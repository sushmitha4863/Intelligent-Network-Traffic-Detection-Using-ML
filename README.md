# Intelligent Network Traffic Detection Using ML

## Overview

This project was developed as a **collaborative team project** to design an Intelligent Intrusion Detection System (IDS) capable of detecting malicious network traffic using Machine Learning algorithms.

The system combines **LightGBM** for offline batch analysis and **Adaptive Random Forest (ARF)** for real-time network traffic detection. A Streamlit-based web interface allows users to upload network traffic datasets and visualize prediction results.
